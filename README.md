# Student Speaking Assessment Report

A simple web application that displays a student’s speaking assessment report similar to IELTS / SpeechAce score reports.  
The project focuses on frontend UI clarity, basic backend data handling, and logical feedback generation.

---

## Tech Stack

- Frontend: React, HTML, CSS, JavaScript
- Backend: Node.js, Express
- Data Source: In-memory JavaScript object
- UI Visualization: CSS progress bars

---

## How to Run the Project

### Backend

```bash
cd backend
npm install
npm run start
```

Backend runs on

```bash
http://localhost:3000
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Backend runs on

```bash
http://localhost:5137

```

This data is returned through a single API endpoint:

```bash
GET /api/report

```

---

### How Feedback Logic Works:

Feedback is generated dynamically based on score ranges using conditional logic.

#### Score Ranges:

Score ≥ 8
→ Excellent performance.

Score ≥ 7 and < 8
→ Very good performance.

Score ≥ 6 and < 7
→ Good performance.

Score < 6
→ Needs improvement.

---

### Deployment:

Frontend: Vercel
Backend: Render
